# Minimal Tetris Game

## Request Changes
    - Score for completed lines
    - Details block - gradient or rounded edges
    - Slow down block fall speed

A simple Tetris cone built with Python and Pygame. Use the keyboard to move and rotate blocks to complete lines.

## How to run

1. Make sure you have Python 3 and Pygame installed
2. Create .venv and active env
3. Run the game - py tetris.py in terminal

```
python tetris.py
```
